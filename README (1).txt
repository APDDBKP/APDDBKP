1. Запустить файл DocGen (при необходимости - от имени администратора).
2. Выбрать файл, содержащий таблицу с исходными данными (в данной версии программы таблица должна быть без строки-заголовка), имеющую формат XLSX либо CSV (разделитель - точка с запятой).
3. Выбрать файл с расширением .docx, содержащий шаблон (template.docx).
4. Ждать генерации файлов.
5. Об успешном окончании работы программы сообщит соответствующее всплывающее окно.
6. Итоговые файлы будут находиться в той же директории, что и запущенный файл DocGen.
6. В случае возникновения ошибок проверить исходные данные и правильность выполнения вышеуказанной последовательности действий.
